How It Works

  1. A list of toons for each team is culled from Knate's Master Raid Team List spreadsheet.
  2. We loop through the list of teams, and gather rogression data for each toon using the Battle.net API.
  3. Using this progression data for the toons, progression for the team is built using the last-killed time for each boss in both Highmaul and Blackrock Foundry. For each boss, if five or more toons have the same last-killed time, then that kill is considered a progression kill for the team as a whole.
  4. This progression data is regenerated every morning at 2AM Arizona time. Prior progression for a team is never erased, only updated, so a team will never go backwards in progression.
  5. Teams are ordered on the display page using the following order: M BRF > H BRF > N BRF > M HM > H HM > N HM. This does mean that teams that have cleared more of N BRF than a team that hasn't cleared any of BRF, even if that team has cleared all of mythic HM.
